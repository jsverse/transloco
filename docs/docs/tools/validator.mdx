This package provides validation for translation files. It validates that the JSON is valid and that it doesn't contain duplicates keys.

### Installation

- Set up [husky](https://github.com/typicode/husky) and [lint-staged](https://github.com/okonet/lint-staged#examples)
- Run `npm i @ngneat/transloco-validator --save-dev`

### Usage
```json file="package.json"
{
  "husky": {
    "hooks": {
      "pre-commit": "lint-staged"
    }
  },
  "lint-staged": {
    "src/assets/i18n/*.json": ["transloco-validator [options]"]
  }
}
```

This will make sure no one accidentally pushes an invalid translation file.


### Options

#### Interpolation forbidden characters for parameter names.

- `--interpolationForbiddenChars`

  `type`: `string`

  `default`: `{}`
